Detailed Description

template<typename T, unsigned int vector_length>
class DevaFramework::BaseVector< T, vector_length >

A generic class representing a column vector.

Template Parameters
T- the type of data in the vector
vector_length- the number of values in the vector. For a column vector, this means the number of rows.
