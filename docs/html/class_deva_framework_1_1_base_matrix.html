Detailed Description

template<typename T, unsigned int n_vectors, unsigned int vector_length>
class DevaFramework::BaseMatrix< T, n_vectors, vector_length >

A generic class, representing an NxM matrix.

Every matrix is to be seen as n_vectors sets of vector_length values, where values in a set and individual sets are stored consecutively. This allows a matrix to be seen as either row-major or column-major

Template Parameters
T- the type of the matrix. On theory, a matrix can be of any type that supports all of the following:
  • Default construction
  • Binary * (star) operator
  • Copy and Move assignment

  • String conversion via std::to_string

n_vectors- The number of sets in the matrix. Every set represents a column.
vector_length- The number of values in a set.
